# detectAsyncLeaks

- **Type:** `boolean`
- **CLI:** `--detectAsyncLeaks`, `--detect-async-leaks`
- **Default:** `false`

Detect asynchronous resources leaking from the test file.

Note that enabling this option will make your tests run much slower. Use only when debugging or developing tests.
